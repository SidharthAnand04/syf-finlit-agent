# Documentation

This folder contains the handoff documentation for the Synchrony Financial Literacy Assistant.

Start here:

- `backend/README.md` - backend architecture and runtime behavior.
- `frontend/README.md` - frontend app structure and admin UI.
- `data/README.md` - Supabase RAG, local KB files, and ingestion.
